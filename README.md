# DiscoBot
A simple project I have made as a request for my school disco.

Starts a bot that accepts songs and sends them to the specific channel.

To create database run:
`python song_database_setup.py`

To start bot run: `python main.py`

Requirements:
- SQLAlchemy
- python-telegram-bot
